**Command-line tools for working with Issues-FS repositories.**

## Installation

```bash
pip install issues-fs-cli
```

## Development

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements-test.txt
pip install -e .
```

## Tests

```bash
pytest -q
```
